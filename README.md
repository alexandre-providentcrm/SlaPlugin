#SLA Plugin - SugarCRM 7.8

####How to use

Import the SlaCalculation class and the library folder to your project and start to use them.

## Features
 * TimeZone
 * Overall Due Date
 * Skip Saturday
 * Skip Sunday
 * Skip Bank Holidays (UK)
 * Time Left

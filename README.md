#SLA Plugin - SugarCRM 7.8

####How to use

Import the SlaCalculation class plus the folde lib to you project and start to use.

## Features
 * TimeZone
 * Overall Due Date
 * Skip saturday
 * Skip Sunday
 * Skip Bank Holidays (UK)
 * Time Left
